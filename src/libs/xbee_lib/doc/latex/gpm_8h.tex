\subsection{Detailed Description}
General Purpose Memory interface, used on multiple module types (900\+HP, 868\+LP, Wi-\/\+Fi). 

Access is done by sending explicit A\+PI frames to the M\+E\+M\+O\+R\+Y\+\_\+\+A\+C\+C\+E\+SS cluster (0x23) of the D\+I\+G\+I\+\_\+\+D\+E\+V\+I\+CE endpoint (0x\+E6) of the target node, either in a self-\/addressed frame, or a frame addressed to a remote node.
