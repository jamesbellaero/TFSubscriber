Header for implementation of Zig\+Bee Cluster Library. 

Type definitions and frame formats.

Based on document 075123r01\+ZB, updated for 075123r02\+ZB

Even though the X\+Bee modules uses big-\/endian byte order for its A\+PI, the payload data should be in the correct format for Rabbit (little-\/endian). Multi-\/byte variables and structure members use an \char`\"{}\+\_\+le\char`\"{} suffix if the value is little-\/endian, a \char`\"{}\+\_\+be\char`\"{} suffix if the value is big-\/endian, and no suffix if the value is in host byte order.
