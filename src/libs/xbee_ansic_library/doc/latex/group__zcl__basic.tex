If \#\+Z\+C\+L\+\_\+\+F\+A\+C\+T\+O\+R\+Y\+\_\+\+R\+E\+S\+E\+T\+\_\+\+FN is defined, the Basic Server Cluster will support the optional \char`\"{}\+Reset to Factory Defaults\char`\"{} command.
